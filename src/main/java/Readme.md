# Instruction to use the application
* use browser open http://localhost:8080/
* Eine Webseite zum Bücher Verwalten 
* Die Webseite ermöglicht es, Bücher hinzuzufügen, zu bearbeiten und zu löschen.
* Die Webseite ermöglicht es Genre anzulegen falls in den Demo Daten was nicht vorhanden ist.
* Die Daten werden in der Datenbank MGIN_Aufgabe_8 gespeichert.
